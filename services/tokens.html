    Tokens

    This service is used to retrieve and use one or many Tokens.

    • find( $identifier )

    #find( $identifier )

    Returns an Token

    ArgumentAcceptsDescription
    $identifierstring, integer, TokenA unique Token identifier
    • Twig
    • PHP

    Returns a Token Query.

    ArgumentAcceptsDescription
    $criteriaarrayAn array of Token Query criteria.

    • PHP

    id: 1
}) %}
